# Autonomous AI Data Analyst

Hey! Welcome to my AI Data Analyst project.

I'm building this project to learn how real-world AI systems work not just theory, but actually shipping something I'm proud of.
 
The idea is simple: upload any dataset, and let AI do the heavy lifting.
No SQL. No coding required from the user. Just ask questions and get answers.

## What it does so far

- Upload any CSV file and instantly see your data
- Automatically shows rows, columns, missing values and memory usage
- Browse your data with an interactive preview
- See column-by-column breakdown of data types and null values

## What's coming next

I'm building this phase by phase:

- ✅ Phase 1 — Upload & Preview (done!)
-  Phase 2 — Auto dataset understanding
-  Phase 3 — Ask AI questions about your data
-  Phase 4 — AI generates and runs Python code for you
-  Phase 5 — Business insight generation

## How to run it yourself
bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/ai-data-analyst.git
cd ai-data-analyst

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the app
python -m streamlit run app.py

Then open your browser at `http://localhost:8501` and upload a CSV!
